# Activity Classification from Smartphone Sensor Data

Phyphoxアプリで取得したスマートフォンの加速度データから、3種類の動作を分類するプロジェクト。

## クラス
- Still（静止）
- Walk（歩行）
- Run（走行）

## データ取得
- アプリ：Phyphox
- センサー：3軸加速度（重力込み）
- サンプリング周波数：100Hz

## 進捗
- [x] データ取得
- [x] 可視化
- [x] 前処理（最初/最後3秒のトリミング）
- [x] 特徴量抽出
- [x] 機械学習モデルの学習
- [x] 評価・考察

## 結果

### モデル：Random Forest（100本の決定木）
### 精度：時系列分割で 100% (Accuracy)

### 主要な特徴量（重要度トップ3）
1. y_mean (0.13)
2. y_min (0.11)
3. x_std (0.09)

### 考察
- 3クラス（Still / Walk / Run）は加速度の振幅が物理的に大きく異なるため、
  シンプルな統計特徴量で完全に分類可能だった
- ただし、Y軸の特徴量が上位を占めていることから、
  モデルは「スマホのポケット内での向き」を判別材料にしている可能性がある
- 汎化性能を上げるには、向きに依存しない特徴量（例：合計加速度の振幅）の追加や、
  異なる被験者・異なる向きでのデータ取得が必要
